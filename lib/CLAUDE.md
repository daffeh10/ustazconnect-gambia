# UstazConnect Gambia

## Project Overview
A tutoring marketplace connecting Quranic teachers (ustazs) with families in The Gambia.

## Tech Stack
- Next.js 15 with App Router
- TypeScript
- Supabase (database)
- Tailwind CSS

## Current Status
- [x] Project setup complete
- [x] Supabase connected
- [x] Database tables created
- [ ] Homepage
- [ ] Ustaz registration
- [ ] Ustaz directory
- [ ] Contact form

## Supported Locations
Serrekunda, Banjul, Bakau, Brikama, Kololi, Kotu, Fajara

## Subjects Offered
Quran Reading, Tajweed, Hifz (Memorization), Arabic Language, Islamic Studies
